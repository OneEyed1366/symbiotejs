Short, direct answers to "how do I do X" — each page is a lookup, not a
tutorial. For the concept behind an answer, follow the link into
[Learn](/symbiote/docs/learn/react/) or [API](/symbiote/docs/api/); for a full copyable app, see
[Examples](/symbiote/docs/examples/).

## Available how-tos

- [Style a component](/symbiote/docs/howtos/styling/) — `StyleSheet.create` vs a Vue SFC
  `<style>` block.
- [Handle press/change events](/symbiote/docs/howtos/events/) — React callback props,
  Vue emits, Angular `@Output()`.
- [Two-way bind a value](/symbiote/docs/howtos/two-way-binding/) — Vue `v-model`,
  Angular `[(value)]`.
- [Share content across surfaces](/symbiote/docs/howtos/portals-and-tunnels/) —
  `createPortal`/`Teleport` vs `createTunnel`.
- [Write platform-specific code](/symbiote/docs/howtos/platform-code/) — the
  `X.ios.ts`/`X.android.ts`/`X/shared.ts` folder convention.
- [Wrap a third-party native view](/symbiote/docs/howtos/third-party-views/) — why the
  library's React component can't be reused, and what can.
- [Turn on diagnostic logging](/symbiote/docs/howtos/debugging/) — `dlog`/`isDebug` and
  the `DEBUG` env var.
